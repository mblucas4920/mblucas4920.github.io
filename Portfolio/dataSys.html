Attached is a PDF file for a data model design project I completed for ITEC370 (Database Systems in Information Technology).

In this project I studied and created many entity-based-relationship diagrams. The project required for me to find a website with an ordering feature and create EDR diagrams to visualize the entities and relationships that exist within the database.

This project helped me understand the importance of visualizing data.
