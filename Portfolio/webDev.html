Attached is a link to a website I created for a project in ITEC362 (Introduction to Web Systems) as well as two PDFs containing my initial wire frames sketch for the site, and a mock-UI I designed to base my website off of.

This project is a "fan-site" utilizing HTML, CSS, and JavaScript to create a stylish and informative website that functions on most standard web browsers. The website is comprised of 15 different webpages, with 4 of them acting as main features and the other 11 being used within an iFrame.

Before creating this website, I had never utilized wireframes or mock-UIs and they both proved to be useful tools. Creating wireframes for my website made me consider the layout of each webpage, as well as the navigation between each page. I designed the mock-UI using Figma and based it off my wireframes. This design became my main reference while coding the CSS for the website. This project taught me the importance of planning.

Working on this project helped me enhance non-technical abilities as it required a great deal of time management, creative thinking, and a willingness to learn, alongside previous HTML knowledge. This valuable experience will help in achieving my post-graduate goal of working with a company that allows me to pursue my interest in website design and development.
