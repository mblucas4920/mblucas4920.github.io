ITEC242: Business Communications

Building_a_PC.pdf

Attached is a PDF for an instruction manual on how to build a PC (personal computer) that I created for ITEC242 (Business Communications).

In this project I utilized the 'assembly guide' instructional format structure. Designing and compiling this guide required strong communication skills and an understanding of how to create a readable and accessible design.

This project allowed for me to take a topic I was already knowledgeable on and turn it into a helpful guide. Creating this guide helped me understand how I can use my knowledge to help others in a productive and professional way.
